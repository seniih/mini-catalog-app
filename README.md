# Mini Catalog

Fake Store API'den ürün çeken, detay sayfası ve basit bir sepet sistemi olan Flutter uygulaması.

## Kullanılan Sürümler

- Dart SDK: ^3.11.0
- http: ^1.6.0

## Kurulum ve Çalıştırma (Installation & Run)

```bash
git clone https://github.com/seniih/mini-catalog-app
cd mini-catalog-app

flutter pub get
flutter run
```

Emulatör ya da bağlı bir cihaz açık olmalı. `flutter devices` ile kontrol edebilirsin.

## Not

Veri kaynağı olarak kullanılan Fake Store API'deki içerikler (ürün adları ve açıklamalar) İngilizce olduğundan uygulamanın arayüzü de İngilizcedir.
